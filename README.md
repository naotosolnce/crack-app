# CRACK! 🪟💥

Pick the lighter side — or the glass breaks!

A party game that sparks conversations. 8 languages, 40+ animals, Wikipedia-sourced weights.

## Files

```
index.html    — Full app (single file, no build needed)
manifest.json — PWA manifest
sw.js         — Service worker (offline support)
icon-192.png  — App icon 192x192
icon-512.png  — App icon 512x512
```

## Deploy

### Vercel (easiest)
1. Create folder with all files
2. `npx vercel` — done

### Netlify
1. Drag the folder into netlify.com/drop

### GitHub Pages
1. Push files to repo
2. Settings → Pages → Deploy from main branch

### Cloudflare Pages
1. Connect repo or direct upload

## Install as App
- **iOS**: Safari → Share → Add to Home Screen
- **Android**: Chrome → Menu → Add to Home Screen
- **Desktop**: Chrome address bar → Install icon

## Game Rules
- Two groups of animals appear
- Pick the LIGHTER side to survive
- Wrong = glass breaks, game over → retry
- Right = next round, keep going
- How far can you go?

## Data Sources
All animal weights from Wikipedia (mean adult values).
See in-game source citations after each answer.
